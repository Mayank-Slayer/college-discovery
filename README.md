# College Discovery Platform

A modern web application to discover, compare and explore colleges.

## Features

- Search Colleges
- Filter by Location
- Compare Colleges
- College Details Page
- Admin Login
- Add College
- Delete College
- MongoDB Integration

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Prisma
- MongoDB

## Installation

```bash
npm install
npm run dev
```
